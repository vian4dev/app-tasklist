## Instalação
Clone o repositório.
~~~
git clone git@github.com:vian4dev/app-tasklist.git
~~~
Acesse o diretório do projeto.
~~~
cd app-tasklist
~~~
Instale as dependências.
~~~
yarn
~~~
Execute a aplicação.
~~~
yarn start
~~~
